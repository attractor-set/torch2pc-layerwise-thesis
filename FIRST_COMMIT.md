# Первый коммит

[English version](FIRST_COMMIT_EN.md)

Первый коммит предназначен для фиксации исходной структуры исследования до
получения экспериментальных результатов. В него входят код, конфигурации,
методологические документы, тесты и средства воспроизводимости.

Рекомендуемое сообщение коммита:

```text
research: initialize neutral reproducible thesis repository
```

Перед фиксацией необходимо выполнить проверки нейтральности формулировок,
языковых пар, локальных ссылок, тестов и структуры репозитория:

```bash
python scripts/check_epistemic_language.py
python scripts/check_language_structure.py
python scripts/check_local_links.py
PYTHONPATH=src pytest -q
./scripts/validate_repository.sh
```

В первый коммит не добавляются локальные переменные окружения, скачанные
датасеты, checkout стороннего репозитория, сырые попытки, checkpoints, PDF
публикаций и приватные материалы.

После коммита рекомендуется создать tag только при необходимости. Первый
научный milestone создается после выполнения C0/C1, а не в момент добавления
каркаса проекта.

## Граница первого коммита

Успешная статическая проверка означает только согласованность структуры,
синтаксиса и тестируемых вспомогательных функций. Она не считается
экспериментальным результатом и не заменяет Docker/ROCm runtime, C0/C1, pilot
или final.
