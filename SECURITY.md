# Политика безопасности

[English version](SECURITY_EN.md)

Проект является исследовательским программным обеспечением и не предназначен
для развертывания как сетевой сервис.

Уязвимости зависимостей, workflow или исполнения произвольного кода следует
сообщать через приватный Security Advisory после публикации репозитория.

Экспериментальные контейнеры работают без сети после подготовки ресурсов.
Перед релизом проверяются сторонние GitHub Actions, базовый Docker-образ и его
digest.
