# Пакет научной статьи

[English version](README_EN.md)

До получения final results структура статьи считается предварительной.
Возможные направления анализа:

1. структурная и численная проверка реализации;
2. зависимость градиентного сходства от inference budget;
3. сопоставление выходных и внутренних представлений;
4. качество и вычислительная стоимость;
5. устойчивость и частота неудачных запусков.

Итоговая публикационная история выбирается после основного анализа, без
исключения результатов, не соответствующих исходным ожиданиям.

Англоязычные исходники:

- `manuscript_EN.tex`;
- `supplementary_EN.tex`.
