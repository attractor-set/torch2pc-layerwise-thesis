# Каталог данных

[English version](README_EN.md)

Наборы данных загружаются локально командой `make prepare` и исключены из Git.

Этап подготовки сохраняет:

- список файлов;
- размер каждого файла;
- SHA-256;
- версию набора данных, доступную через используемую библиотеку;
- параметры преобразования.

Данные не загружаются во время финального эксперимента.
