# ADR-001: Использовать Torch2PC как основную реализацию

[English version](ADR-001-use-torch2pc_EN.md)

Статус: принято.

## Контекст

Исследование сравнивает режимы Exact, FixedPred и Strict из линии работ
Rosenbaum.

## Решение

Использовать полный зафиксированный commit Torch2PC, проверить его по поправке
2025 года и блокировать pilot/final до прохождения C0/C1.

## Последствия

Результаты относятся к конкретной версии реализации и заявленным архитектурам,
а не автоматически ко всем алгоритмам предиктивного кодирования.
