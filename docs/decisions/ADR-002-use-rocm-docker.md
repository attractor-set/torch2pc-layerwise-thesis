# ADR-002: Использовать native Ubuntu и ROCm Docker

[English version](ADR-002-use-rocm-docker_EN.md)

Статус: принято.

## Контекст

Основная GPU - Radeon RX 7700 XT. Время и численное поведение должны
сравниваться в контролируемой среде.

## Решение

Финальные GPU-запуски выполняются в нативном Ubuntu и версионированном
ROCm/PyTorch образе. Аналитический контроль выполняется на CPU float64.

## Последствия

Контейнер не фиксирует драйвер хоста, ядро, BIOS, частоты, температуру и
фоновую нагрузку. Они регистрируются отдельно.
