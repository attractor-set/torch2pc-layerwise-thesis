# ADR-002: использовать непосредственно установленную Ubuntu и контейнер Docker с ROCm

[English version](ADR-002-use-rocm-docker_EN.md)

Статус: принято.

## Контекст

Основное вычислительное устройство — Radeon RX 7700 XT, основная система —
непосредственно установленная Ubuntu 24.04.4. [Время выполнения](../glossary.md#term-runtime) и
численное поведение должны сравниваться в контролируемой среде.

## Решение

Итоговые запуски на GPU выполняются в непосредственно установленной Ubuntu и
версионированном образе ROCm/PyTorch. Аналитический контроль выполняется на CPU
с типом данных float64.

## Последствия

Контейнер не фиксирует драйвер узла, ядро, BIOS, частоты, температуру и фоновую
нагрузку. Эти параметры регистрируются отдельно.
