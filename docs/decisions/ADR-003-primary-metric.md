# ADR-003: макро-F1 как основная метрика выбора

[English version](ADR-003-primary-metric_EN.md)

Статус: принято до пилотного исследования.

Макро-F1 используется для выбора по валидационной выборке и для основного
итогового сравнения. Точность публикуется как вторичная метрика. Решение не
предполагает, что макро-F1 обязательно обнаружит различия; оно фиксирует
правило выбора до доступа к тестовой выборке.
