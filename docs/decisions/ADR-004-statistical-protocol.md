# ADR-004: парный статистический протокол

[English version](ADR-004-statistical-protocol_EN.md)

Статус: принято до пилотного исследования.

- единица анализа — независимо обученная модель;
- методы сопоставляются по `model_seed`;
- для основного анализа требуется не менее 10 полных пар;
- публикуются исходные значения, размер эффекта и интервал;
- различие и эквивалентность оцениваются раздельно;
- применяется поправка Холма;
- неудачные запуски остаются в реестре и входят в отчёт о доле успешных
  запусков.
