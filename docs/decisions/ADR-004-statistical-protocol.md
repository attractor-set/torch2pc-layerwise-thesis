# ADR-004: Парный статистический протокол

[English version](ADR-004-statistical-protocol_EN.md)

Статус: принято до pilot.

- единица анализа - независимо обученная модель;
- методы сопоставляются по model seed;
- минимум 10 complete pairs для основного анализа;
- публикуются исходные значения, эффект и интервал;
- различие и эквивалентность оцениваются раздельно;
- применяется Holm adjustment;
- failed runs остаются в реестре и входят в success-rate отчет.
