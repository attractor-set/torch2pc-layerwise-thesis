# ADR-027: фиксация протокола описательного анализа сопоставленного профилирования

[English version](ADR-027-stage3b-matched-descriptive-analysis-protocol_EN.md)

- **Статус:** принято
- **Дата:** 2026-07-21

## Контекст

Запечатанные [доказательные материалы](../glossary.md#term-evidence)
сопоставленного профилирования содержат 288 ячеек, 96 сопоставленных блоков,
1440 строк повторов и полный поток событий локальности. Неизменяемый тег и
черновой релиз уже связаны с коммитом слияния
`21ddfb8840674871f0b9d888b36397f5cf0e111b`.

Сбор данных завершён, но сравнительный анализ ещё не выполнялся. До вычисления
отношений времени, Парето-множеств и решений по B1/B2 необходимо закрыть
степени свободы анализа. При этом нельзя представлять такую фиксацию как
предварительную регистрацию до сбора данных.

В репозитории уже существует ранняя частичная реализация
`stage3b_matched_analysis.py`. Она не считается авторизованной реализацией
этого протокола и должна быть приведена в соответствие в отдельном PR.

## Решение

Создать детерминированный пакет:

```text
experiments/frozen/stage3b-matched-descriptive-analysis-v1/
  protocol.json
  SHA256SUMS
```

Протокол фиксирует:

- источник и контрольные суммы запечатанных [доказательных материалов](../glossary.md#term-evidence);
- независимую единицу `model_seed` и запрет псевдорепликации;
- неизменяемый порядок агрегации;
- основную, вторичные и структурные оценки;
- описательную модель масштабирования для семи явно перечисленных отношений;
- семимерное правило
  [Парето-допустимости](../glossary.md#term-pareto-admissibility) без скрытой
  скаляризации;
- пороги 15% для `FixedPred`, 20% для `Strict`, 3% для максимальной регрессии и
  15% для роста памяти;
- формальное решение `retain / conditional / reject_or_revise`;
- правила пропусков, чувствительности и точный набор из 18 будущих файлов;
- закрытую границу выполнения, публикации, `EX-IF0` и активации политики.

Построитель протокола может проверять только идентичность, структуру и
категориальную матрицу входа. Он не использует наблюдённые значения времени,
памяти или локальности для выбора правил.

## Следствия

Положительные:

- сравнительные решения становятся машинно воспроизводимыми;
- зарегистрированные ранее пороги не меняются после сбора данных;
- повторы и конфигурации не увеличивают независимое `n=3`;
- смешанный результат может быть выражен как `conditional`, а не принудительно
  сведён к одному победителю;
- исходные доказательные материалы остаются только для чтения.

Ограничения:

- фиксация выполняется после сбора и до анализа, а не является предварительной
  регистрацией до сбора данных;
- при трёх `model_seed` запрещены сильные статистические утверждения;
- семимерное Парето-множество может содержать несколько альтернатив;
- ранняя частичная реализация анализа должна быть заменена или расширена в
  отдельном `PR` реализации.
