# Настройка GitHub

[English version](github-setup_EN.md)

1. Проверить данные автора и адрес репозитория в `CITATION.cff`,
   `pyproject.toml` и LaTeX.
2. Выполнить `./scripts/bootstrap_github.sh`.
3. Создать пустой удалённый репозиторий.
4. Отправить ветку `main`.
5. Включить задачи, обсуждения, проекты и защиту ветки.
6. Сделать проверки `ci`, `docs` и `thesis-build` обязательными.
7. Создать проект GitHub с полями:
   - тип;
   - приоритет;
   - глава диссертации;
   - публикационный охват;
   - статус.
8. После первого публичного выпуска связать репозиторий с Zenodo.
