# Настройка GitHub

[English version](github-setup_EN.md)

1. Проверить данные автора и URL репозитория в `CITATION.cff`, `pyproject.toml` и LaTeX.
2. Выполнить `./scripts/bootstrap_github.sh`.
3. Создать пустой удаленный репозиторий.
4. Отправить ветку `main`.
5. Включить Issues, Discussions, Projects и защиту ветки.
6. Сделать проверки `ci`, `docs` и `thesis-build` обязательными.
7. Создать GitHub Project с полями:
   - тип;
   - приоритет;
   - глава диссертации;
   - публикационный охват;
   - статус.
8. После первого публичного релиза связать репозиторий с Zenodo.
