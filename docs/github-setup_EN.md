# GitHub repository setup

[Русская версия](github-setup.md)


1. Verify the author metadata and repository URL.
2. Run `./scripts/bootstrap_github.sh`.
3. Create the remote repository.
4. Push `main`.
5. Enable Issues, Discussions, Projects, and branch protection.
6. Require the `ci`, `docs`, and `thesis-build` checks.
7. Create a GitHub Project with Type, Priority, Thesis chapter, Publication
   scope, and Status fields.
8. Link the repository to Zenodo after the first public release.
