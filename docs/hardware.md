# Аппаратная и программная среда

[English version](hardware_EN.md)

Целевой узел: RX 7700 XT, Ryzen 7 5700X3D, 64 ГБ RAM, native Ubuntu.
Конкретные версии kernel, amdgpu, ROCm, PyTorch, Docker image и BIOS-настройки
записываются при первом runtime-запуске и не предполагаются успешными заранее.

CPU float64 используется для строгих контрольных проверок. GPU FP32 используется
для основного обучения. Timing выполняется отдельно, без фоновой GPU-нагрузки,
с warm-up и синхронизацией.
