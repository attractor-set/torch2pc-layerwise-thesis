# Аппаратная и программная среда

[English version](hardware_EN.md)

Целевой узел: RX 7700 XT, Ryzen 7 5700X3D, 64 ГБ оперативной памяти и
непосредственно установленная Ubuntu. Конкретные версии ядра, `amdgpu`, ROCm,
PyTorch, образа Docker и настроек BIOS записываются при первом запуске
измерения времени и не считаются успешными заранее.

CPU с типом данных float64 используется для строгих контрольных проверок. GPU
с типом данных float32 используется для основного обучения. Измерение времени
выполняется отдельно, без фоновой нагрузки GPU, с разогревом и синхронизацией.
