# План публикации

[English version](publication-plan_EN.md)

Публикационная история будет определена после основного анализа. Предварительно
рассматриваются следующие направления:

1. correction-aware контроль реализации;
2. зависимость градиентного сходства от inference budget;
3. различие между сходством выходов и внутренних представлений;
4. quality/compute trade-off;
5. устойчивость и частота неудачных запусков.

Название, abstract и claims не фиксируются до получения final results. Слово
`впервые` не используется без отдельного систематического поиска. Публичность
репозитория согласуется с требованиями будущей площадки и double-blind review.
