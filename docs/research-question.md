# Исследовательский вопрос

[English version](research-question_EN.md)

## Наблюдаемая проблема

Практические реализации predictive coding могут отличаться по формуле ошибки,
режиму обновления, бюджету inference, численной точности и вычислительной
стоимости. Сравнение только итоговой accuracy не описывает послойные сигналы и
стоимость получения результата.

## Вопрос

При каких зафиксированных условиях режимы Torch2PC дают наблюдения, близкие к
BP по качеству, градиентам и представлениям, а при каких условиях различия
выходят за заранее заданные границы?

## Граница исследования

Работа наблюдает конкретный commit Torch2PC на LeNet-подобных архитектурах и
выбранных наборах данных. Она не предполагает вывод о всех вариантах predictive
coding и не оценивает биологическую правдоподобность напрямую.
