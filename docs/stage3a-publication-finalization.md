# Завершение публикации этапа 3A

[English version](stage3a-publication-finalization_EN.md)

Скрипт `scripts/finalize_stage3a_publication.py` синхронизирует статусную
документацию с завершённой послойной кампанией, заменяет агрегатор на
реализацию, учитывающую метаданные, создаёт двуязычные записи завершения и
добавляет примечание об исправлении происхождения артефактов без изменения
первичных данных.

Исходный `COMPOSE_RESOLVED.yaml` сохраняется: исправление документирует
устаревшую переменную, не переписывая исторический снимок.

После запуска необходимо повторно выполнить агрегирование из каталога исходных
данных, проверить число строк и хэши, затем добавить публикационные файлы в
Git.
