# Ноутбуки

[English version](README_EN.md)

`analysis/` содержит ноутбуки только для анализа зарегистрированных
артефактов. Они не должны изменять экспериментальные конфигурации или выполнять
скрытую настройку гиперпараметров.

`legacy/full_pipeline_v11.ipynb` является замороженной точкой миграции. Он
сохраняется для проверки полноты переноса логики, но не является основным
оркестратором версии v0.1.0.

Новые научные функции размещаются в `src/torch2pc_thesis/`.
