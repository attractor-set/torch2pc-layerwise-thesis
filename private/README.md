# Приватные локальные материалы

[English version](README_EN.md)

Каталог исключен из Git и может содержать:

- PDF научных статей;
- комментарии научного руководителя;
- черновые рецензии;
- приватные версии текста;
- временные сравнительные материалы;
- данные, распространение которых ограничено.

Публичные выводы должны ссылаться на библиографические записи, DOI и
воспроизводимые артефакты, а не на локальный путь к PDF.
