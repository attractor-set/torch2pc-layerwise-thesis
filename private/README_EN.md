# Private local materials

[Русская версия](README.md)

The directory contents are excluded from Git except for `README.md` and
`README_EN.md`. Locally, the directory may contain:

- scientific article PDFs;
- supervisor comments;
- internal reviews;
- private manuscript drafts;
- temporary comparison materials;
- data with restricted redistribution conditions.

Public conclusions must rely on bibliographic records, DOI references,
registered experiments, and reproducible manifests rather than local PDF paths
or private notes.
