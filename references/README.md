# Литература

[English version](README_EN.md)

Публичный каталог содержит библиографические данные и трассировку источников,
но не повторно распространяемые PDF.

Локальные PDF можно хранить в `references/pdfs/`, который исключен из Git.

Перед публикацией необходимо:

- проверить каждую BibTeX-запись по первичному источнику;
- убедиться, что каждая запись библиографии содержательно цитируется;
- связать источник с гипотезой, методом, набором данных или ограничением;
- удалить неиспользуемые и дублирующиеся версии.
