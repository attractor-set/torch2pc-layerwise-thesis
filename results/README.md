# Результаты

[English version](README_EN.md)

В Git включаются только агрегированные наблюдения, таблицы, рисунки и компактные
манифесты. Сырые попытки и checkpoints остаются локальными. Ни один файл в этом
каталоге не считается выводом без ссылки на зарегистрированные run IDs.

- Stage 1: `results/summaries/`, `results/tables/`, `results/figures/`;
- Stage 2: `results/stage-2/`;
- Stage 3: `results/stage-3/`, отдельная provenance chain без placeholder metrics.
