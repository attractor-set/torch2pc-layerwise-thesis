# Results

[Русская версия](README.md)

Only aggregate observations, tables, figures, and compact manifests are stored
in Git. Raw attempts and checkpoints remain local. No file is treated as a
result without registered run IDs.

- Stage 1: `results/summaries/`, `results/tables/`, `results/figures/`;
- Stage 2: `results/stage-2/`;
- Stage 3: `results/stage-3/`, with a separate provenance chain and no placeholder metrics.
