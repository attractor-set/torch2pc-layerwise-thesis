# Результаты Stage 3

[English version](README_EN.md)

Каталог зарезервирован для новой provenance chain Stage 3. До начала исполнения
здесь не публикуются synthetic результаты или placeholder metrics.

Планируемые подкаталоги: `profiling/`, `pilot/`, `accelerator-screening/`, `final/`. Raw runs и
checkpoints исключаются из Git; компактные gates, summaries, tables, figures и
manifests добавляются отдельными evidence commits.
