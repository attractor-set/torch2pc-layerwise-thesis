# Тесты научной корректности

[English version](README_EN.md)

Runtime-тесты этого каталога требуют подготовленного checkout Torch2PC.

После первого доверенного запуска CPU float64 сюда добавляются регрессионные
тесты по:

- аудиту поправки;
- C0 Exact против BP;
- C1 FixedPred против Exact;
- численным допускам;
- структуре послойных результатов.
